# 课件文件部署配置说明

## 重要提示

**产教结合页面必须在部署了AI代理服务的域名下才能正常工作！**

原因：
- 产教结合页面需要调用 `/api/` 后端服务
- `/api/` 需要 Nginx 代理到 Node.js 服务（端口3000）
- Node.js 服务处理火山引擎API调用

## 部署方案

### 方案1：课件和产教结合部署在同一域名（推荐）

如果课件和产教结合部署在同一域名下，链接可以使用相对路径：
```html
<a href="../产教结合/index.html">产教融合</a>
```

### 方案2：课件和产教结合部署在不同域名

如果课件部署在其他域名，产教结合部署在有AI功能的域名，链接必须使用绝对URL：

**当前配置的URL：**
```html
<a href="https://ai.projectmath.xyz/产教结合/index.html">产教融合</a>
```

**部署到新域名时需要修改：**
1. 搜索所有课件文件中的 `https://ai.projectmath.xyz/产教结合/index.html`
2. 替换为实际部署AI功能的域名
3. 例如：`https://your-ai-domain.com/产教结合/index.html`

## 批量修改方法

在服务器上执行（替换 YOUR-AI-DOMAIN 为实际域名）：

```bash
cd /usr/share/nginx/html/chanjiaojiehe/课件
find . -name "*.html" -type f -exec sed -i 's|https://ai\.projectmath\.xyz/产教结合/index\.html|https://YOUR-AI-DOMAIN/产教结合/index.html|g' {} \;
```
