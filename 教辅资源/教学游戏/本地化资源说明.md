# 资源本地化说明

为了完全离线使用，需要下载以下资源文件：

## 1. Tailwind CSS
- **文件路径**: `assets/vendor/tailwindcss.min.js`
- **下载方式**: 
  - 访问 https://cdn.tailwindcss.com
  - 右键保存页面内容为 `tailwindcss.min.js`
  - 放到 `assets/vendor/` 目录下

## 2. MathJax
- **文件路径**: `assets/vendor/mathjax/tex-svg.js`
- **下载方式**:
  - 访问 https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-svg.js
  - 保存为 `tex-svg.js`
  - 放到 `assets/vendor/mathjax/` 目录下
  - **注意**: MathJax 可能需要额外的依赖文件，建议下载完整包

## 3. Material Icons Outlined 字体
- **文件路径**: `assets/fonts/MaterialIconsOutlined.woff2`
- **下载方式**:
  - 访问 Google Fonts: https://fonts.google.com/icons
  - 下载 Material Icons Outlined 字体包
  - 提取 `MaterialIconsOutlined-Regular.woff2` 文件
  - 重命名为 `MaterialIconsOutlined.woff2`
  - 放到 `assets/fonts/` 目录下

## 目录结构
```
教辅资源/教学游戏/
├── 高职数学狼人杀v2.html
├── assets/
│   ├── vendor/
│   │   ├── tailwindcss.min.js
│   │   └── mathjax/
│   │       └── tex-svg.js
│   ├── icons/
│   │   └── material-icons-outlined.css
│   └── fonts/
│       └── MaterialIconsOutlined.woff2
```

## 验证
下载完成后，在浏览器中打开 HTML 文件，检查：
1. 控制台无 404 错误
2. 页面样式正常显示
3. 图标正常显示
4. 数学公式正常渲染
