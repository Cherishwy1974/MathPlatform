第一章《函数与初等函数》教案

教学项目

教学项目 函数与初等函数

授课地点 多媒体教室

授课形式 线下教学

学情分析 学生已具备基本的数学基础，但对函数概念的理解还不够深入。学生在函数的概念、性质、基本初等函数、复合函数方面需要重点指导。学生喜欢可视化与案例化教学，可借助现有交互课件和仿真实验激发学习兴趣。

教学目标

知识目标：使学生深入理解函数的概念、掌握函数的基本性质和基本初等函数、熟练运用复合函数。

能力目标：培养学生利用函数解决实际问题的能力、提高数学建模能力。

素质目标：提高学生的数学抽象思维和逻辑推理能力，培养严谨的数学态度。

教学重点 函数的概念、函数的基本性质、基本初等函数、复合函数的处理。

教学难点及应对

难点：函数概念的理解、复合函数的处理、函数性质的证明。

应对策略：通过具体的工程案例演示，分步骤详细讲解，辅以Lab1系列仿真实验和小组讨论，加深学生理解。

教学资源

教材：《高等数学》

媒体资源：课件《第1章函数与初等函数》、Lab1系列仿真实验

环境设备：多媒体教室配备投影仪和电脑

教学方法

讲授法：详细讲解函数的概念和基本性质。

问答法：鼓励学生提问，通过问题引导学生深入思考。

分组练习法：学生分组完成练习题，互相讨论和解答。

演示法：通过Lab1系列软件演示函数过程。

函数的概念（30min）【教师】详细讲解函数的定义
定义1 设$D$是实数集$R$的一个子集，如果对于$D$中的每一个数$x$，按照某种对应法则$f$，都有唯一确定的实数$y$与之对应，则称$f$是定义在$D$上的函数，记作$y = f(x)$。
其中：
- $D$称为函数的定义域
- $f(D) = \{y		y = f(x), x \in D\}$称为函数的值域
- $x$称为自变量
- $y$称为因变量
【教师】讲解函数的表示方法
1. 解析法：用数学表达式表示，如$y = x^2 + 2x + 1$
2. 图像法：用图形表示，如平面上的曲线
3. 表格法：用表格表示，如实验数据
4. 描述法：用文字描述，如"温度随时间的变化"
【教师】使用Lab1-1展示函数的概念
【学生】观察函数的几何表示
例1 判断下列关系是否为函数：
（1）$y = x^2$
（2）$x^2 + y^2 = 1$
（3）$y = \pm\sqrt{x}$
解：
（1）是函数，因为对于每个$x \in R$，都有唯一的$y = x^2 \geq 0$与之对应
（2）不是函数，因为对于$x = 0$，有$y = \pm 1$两个值
（3）不是函数，因为对于$x > 0$，有$y = \pm\sqrt{x}$两个值
例2 求函数$f(x) = \frac{1}{\sqrt{x-1}}$的定义域。
解：要使函数有意义，必须：
1. $x - 1 > 0$，即$x > 1$
2. $\sqrt{x-1} \neq 0$，即$x \neq 1$
综合得定义域为$D = (1, +\infty)$
【学生】完成函数概念练习		深入学习函数的基本概念
函数的基本性质（30min）【教师】详细讲解函数的基本性质
1. 有界性
如果存在常数$M$，使得$		f(x)\leq M$对所有$x \in D$成立，则称$f(x)$有界。
2. 单调性
如果对于$x_1, x_2 \in D$，当$x_1 < x_2$时，有$f(x_1) < f(x_2)$，则称$f(x)$单调递增；
如果对于$x_1, x_2 \in D$，当$x_1 < x_2$时，有$f(x_1) > f(x_2)$，则称$f(x)$单调递减。
3. 奇偶性
如果$f(-x) = f(x)$对所有$x \in D$成立，则称$f(x)$为偶函数；
如果$f(-x) = -f(x)$对所有$x \in D$成立，则称$f(x)$为奇函数。
4. 周期性
如果存在常数$T > 0$，使得$f(x + T) = f(x)$对所有$x \in D$成立，则称$f(x)$为周期函数，$T$称为周期。
【教师】使用Lab1-2展示函数的性质
【学生】观察函数性质的几何表示
例3 判断函数$f(x) = x^2$的性质。
解：
1. 有界性：在有限区间上有界，在无穷区间上无界
2. 单调性：在$(-\infty, 0]$上单调递减，在$[0, +\infty)$上单调递增
3. 奇偶性：$f(-x) = (-x)^2 = x^2 = f(x)$，所以是偶函数
4. 周期性：不是周期函数
例4 判断函数$f(x) = \sin x$的性质。
解：
1. 有界性：$		\sin x\leq 1$，所以有界
2. 单调性：在$[-\frac{\pi}{2}, \frac{\pi}{2}]$上单调递增，在$[\frac{\pi}{2}, \frac{3\pi}{2}]$上单调递减
3. 奇偶性：$f(-x) = \sin(-x) = -\sin x = -f(x)$，所以是奇函数
4. 周期性：$f(x + 2\pi) = \sin(x + 2\pi) = \sin x = f(x)$，所以是周期函数，周期为$2\pi$
【学生】完成函数性质练习		深入掌握函数的基本性质
基本初等函数（35min）【教师】详细讲解基本初等函数
1. 幂函数
$y = x^\alpha$（$\alpha$为常数）
- 当$\alpha > 0$时，函数在$(0, +\infty)$上单调递增
- 当$\alpha < 0$时，函数在$(0, +\infty)$上单调递减
- 当$\alpha = 0$时，$y = 1$（常数函数）
2. 指数函数
$y = a^x$（$a > 0, a \neq 1$）
- 定义域：$(-\infty, +\infty)$
- 值域：$(0, +\infty)$
- 当$a > 1$时，函数单调递增
- 当$0 < a < 1$时，函数单调递减
3. 对数函数
$y = \log_a x$（$a > 0, a \neq 1$）
- 定义域：$(0, +\infty)$
- 值域：$(-\infty, +\infty)$
- 当$a > 1$时，函数单调递增
- 当$0 < a < 1$时，函数单调递减
4. 三角函数
$y = \sin x, \cos x, \tan x, \cot x, \sec x, \csc x$
- $\sin x, \cos x$：定义域$(-\infty, +\infty)$，值域$[-1, 1]$
- $\tan x$：定义域$x \neq k\pi + \frac{\pi}{2}$，值域$(-\infty, +\infty)$
5. 反三角函数
$y = \arcsin x, \arccos x, \arctan x, \text{arccot } x$
- $\arcsin x$：定义域$[-1, 1]$，值域$[-\frac{\pi}{2}, \frac{\pi}{2}]$
- $\arccos x$：定义域$[-1, 1]$，值域$[0, \pi]$
- $\arctan x$：定义域$(-\infty, +\infty)$，值域$(-\frac{\pi}{2}, \frac{\pi}{2})$
【教师】使用Lab1-3展示基本初等函数
【学生】观察基本初等函数的图像
例5 画出函数$y = 2^x$的图像并分析其性质。
解：$y = 2^x$是指数函数
- 定义域为$(-\infty, +\infty)$
- 值域为$(0, +\infty)$
- 单调递增
- 当$x \to -\infty$时，$y \to 0$；当$x \to +\infty$时，$y \to +\infty$
- 图像经过点$(0, 1)$
例6 求函数$y = \log_2(x-1)$的定义域和值域。
解：
- 定义域：$x - 1 > 0$，即$x > 1$，所以$D = (1, +\infty)$
- 值域：由于$\log_2(x-1)$可以取到所有实数值，所以值域为$R = (-\infty, +\infty)$
【学生】完成基本初等函数练习		深入学习基本初等函数
复合函数（25min）【教师】详细讲解复合函数的概念
定义2 设$y = f(u)$，$u = g(x)$，则$y = f[g(x)]$称为复合函数。
其中$u$称为中间变量。
【教师】讲解复合函数的求导
如果$y = f(u)$，$u = g(x)$都可导，则
$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$
这就是链式法则。
【教师】讲解复合函数的定义域
复合函数$y = f[g(x)]$的定义域是使得$g(x)$的值域包含在$f(u)$的定义域内的所有$x$的集合。
【教师】使用Lab1-4展示复合函数
【学生】观察复合函数的构成
例7 求复合函数$y = \sin(x^2)$的导数。
解：设$u = x^2$，则$y = \sin u$
$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = \cos u \cdot 2x = 2x\cos(x^2)$
例8 求复合函数$y = e^{x^2 + 1}$的导数。
解：设$u = x^2 + 1$，则$y = e^u$
$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = e^u \cdot 2x = 2xe^{x^2 + 1}$
例9 求复合函数$y = \ln(\sin x)$的定义域。
解：要使函数有意义，必须$\sin x > 0$
所以定义域为$D = \{x		2k\pi < x < (2k+1)\pi, k \in Z\}$
【学生】完成复合函数练习		深入学习复合函数的概念
